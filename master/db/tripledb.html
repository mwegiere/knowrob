 tripledb_load(+URL, +Options) is semidet
Same as tripledb_load/3 with universal scope, and graph name argument as given in the options list or "user" if none is given.
Arguments:
URL- URL of a RDF file.
Options- List of options.
 tripledb_load(+URL, +Scope, +Graph) is semidet
Load RDF data from file, and assert it into the triple DB using the scope provided, and into the graph with the name provided.
Arguments:
URL- URL of a RDF file.
Scope- The subject of a triple.
Graph- The graph name.
