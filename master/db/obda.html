 obda_add(+Module) is det
Registers a Prolog Module that implements the OBDA interface.
Arguments:
Module- a Prolog module.
 obda_remove(+Module) is det
Removes a previously registered OBDA module.
Arguments:
Module- a Prolog module.
 obda_import(+Directory) is det
Calls import on any registered OBDA module with given directory.
Arguments:
Directory- a Prolog module.
 obda_export(+Directory) is det
Calls export on any registered OBDA module with given directory.
Arguments:
Directory- a Prolog module.
 obda_whipe is det
Whipe all OBDA databases of registered modules.
 access(+DataStatement, +Scope) is nondet
Try accessing a data statement with registered OBDA modules by calling their access predicate.
Arguments:
DataStatement- a term holds(Subject,Property,Value).
Scope- a term of the form QueryScope->FactScope.
