 subscope_of(+Sub, +Sup) is det
True if scope Sup contains all facts that are contained in scope Sub.
Arguments:
Sub- a scope dict.
Sup- a scope dict.
 scope_merge(+A, +B, -Merged) is det
Merge two scopes. The merged scope contains all facts that are contained in scope A or scope B.
Arguments:
A- a scope dict.
B- a scope dict.
 scope_intersect(+A, +B, -Merged) is det
Intersect two scopes. The intersected scope contains all facts that are contained in scope A and also in scope B.
Arguments:
A- a scope dict.
B- a scope dict.
 scope_overlaps_query(+A, -B) is semidet
B is the scope of all facts whose scope overlaps with A.
 scope_update(+Original, +Inserted, -Updated) is det
Update a scope with new values taken from another scope dictionary. In case keys appear in both scopes, only the value from Inserted is used in Updated scope.
Arguments:
Original- a scope dictionary.
Inserted- a scope dictionary.
Updated- a scope dictionary.
 scope_remove(+Original, +Key, -Value, -Updated) is det
Removes a key from a scope dictionary.
Arguments:
Original- a scope dictionary.
Key- a key in the scope dictionary.
Value- the value popped out of the dictionary.
Updated- the scope dictionary without Key.
